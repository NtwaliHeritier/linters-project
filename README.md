# Linter project

This project is a C++ linter that flags basic errors to include readablity.

Listed below are the methods built in this project:

-   `check_trailing` 
-   `check_semi_colon`
-   `check_paranthesis`
-   `check_empty_space` 
-   `check_empty_line`
-   `check_below_line`
-   `check_indentation` 


## Built With
-   Ruby
